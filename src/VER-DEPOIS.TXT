Se eu tenho uma imagem que apliquei filter pelo html, como faço pra baixar ela e não baixar a imagem original? Quando tento salvar a imagem ela vem sem os filters aplicados

Quando você aplica filtros a uma imagem no HTML usando CSS, o filtro é aplicado apenas visualmente na página, e a imagem original não é alterada. Portanto, se você tentar salvar a imagem usando um método padrão como clicar com o botão direito do mouse e selecionar "Salvar imagem como", a imagem salva será a original sem os filtros aplicados.

Para baixar a imagem com os filtros aplicados, você pode usar JavaScript para criar uma nova imagem com os filtros aplicados e, em seguida, salvar essa nova imagem. Aqui está um exemplo de como fazer isso:

javascript
Copy code
function downloadFilteredImg() {
  // Seleciona a imagem original e o canvas
  const img = document.querySelector('.original-img');
  const canvas = document.createElement('canvas');

  // Define as dimensões do canvas como as dimensões da imagem
  canvas.width = img.width;
  canvas.height = img.height;

  // Desenha a imagem com os filtros no canvas
  const ctx = canvas.getContext('2d');
  ctx.filter = 'brightness(150%) contrast(200%)'; // Aplica os filtros
  ctx.drawImage(img, 0, 0);

  // Cria um link de download para a nova imagem
  const link = document.createElement('a');
  link.download = 'filtered-img.png'; // Define o nome do arquivo
  link.href = canvas.toDataURL(); // Define o conteúdo do arquivo como a imagem do canvas
  link.click(); // Clica no link para iniciar o download
}
Nesse exemplo, a função downloadFilteredImg cria um novo canvas e desenha a imagem original com os filtros aplicados no canvas. Em seguida, a função cria um link de download para a nova imagem e inicia o download quando o link é clicado. Observe que os filtros são definidos usando a propriedade filter do contexto do canvas. Você pode alterar os filtros para atender às suas necessidades.

No HTML, você pode chamar a função downloadFilteredImg em um evento de clique, por exemplo:

html
Copy code
<img src="original-img.png" class="original-img">
<button onclick="downloadFilteredImg()">Baixar imagem filtrada</button>
Nesse exemplo, a imagem original é exibida na página com a classe .original-img, e um botão é exibido abaixo da imagem. Quando o botão é clicado, a função downloadFilteredImg é chamada para baixar a imagem filtrada.
